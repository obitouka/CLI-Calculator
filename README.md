# Calculator CLI

A simple command-line calculator built in Python using `argparse`.

## HOW TO USE: 

### Show Help
```bash
python calc.py -h
```

<br>

### Example
```bash
python calc.py --num1 23 --num2 3 --operation +
```
OR 
```bash
python calc.py -n1 23 -n2 2 -op +
```

<br>

### Output:
```
25
```
